What is Bootstrap?

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Key Features
  • Responsive grid system
  • Extensive prebuilt components
  • Powerful plugins built on jQuery
  • Sass variables and mixins
  • Responsive utility classes
  • Mobile-first approach
History

Bootstrap was developed by Mark Otto and Jacob Thornton at Twitter as a framework to encourage consistency across internal tools.

First released as an open-source project on August 19, 2011.

Current version: Bootstrap 5 (released May 2021)

Why Use Bootstrap?
Advantages
  • Speeds up development time
  • Responsive by design
  • Consistent design across browsers
  • Customizable with Sass
  • Extensive documentation
  • Large community support
Considerations
  • Can lead to similar-looking websites
  • Requires learning framework conventions
  • File size overhead if not customized
  • May require overriding styles

Basic Template
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Template</title>
  <link href="bootstrap.min.css" rel="stylesheet">
</head>
<body>
  
  <div class="container">
    <h1>Hello, Bootstrap!</h1>
    <button class="btn btn-primary">Click Me</button>
  </div>
  
  <script src="bootstrap.bundle.min.js"></script>
</body>
</html>

Grid System

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

Grid Classes

The grid system has six responsive breakpoints:

  • .col- - Extra small devices (portrait phones, <576px)
  • .col-sm- - Small devices (landscape phones, ≥576px)
  • .col-md- - Medium devices (tablets, ≥768px)
  • .col-lg- - Large devices (desktops, ≥992px)
  • .col-xl- - Extra large devices (large desktops, ≥1200px)
  • .col-xxl- - Extra extra large devices (≥1400px)

When to Use Bootstrap

Consider Using Bootstrap When...
  • Time is limited: Bootstrap accelerates development with pre-built components
  • Design fits Bootstrap patterns: When your design aligns with Bootstrap's component library
  • You control the HTML: When you have full control over the markup
  • Need multiple components: When you need many UI components (buttons, forms, navbars, etc.)
  • Limited CSS experience: When you need cross-browser consistency without deep CSS knowledge
  • Rapid prototyping: For quickly creating prototypes and MVPs
  • Team collaboration: When working with teams to ensure consistent UI
Bootstrap Package Includes
Scaffolding

Basic structure with Grid System, link styles, and background

CSS

Global CSS settings, fundamental HTML elements styled with extensible classes

Components

Over a dozen reusable components (dropdowns, navigation, alerts, etc.)

JavaScript Plugins

Custom jQuery plugins for interactive components

Consider Custom CSS When...
  • Reasonable deadline: When you have time to write custom CSS
  • Unique design: When your design doesn't fit Bootstrap patterns
  • CMS limitations: When working with CMS that restricts HTML modifications
  • Minimal needs: When you only need one or two Bootstrap features
  • CSS expertise: When you have experience writing custom CSS
  • Performance concerns: When minimizing file size is critical
  • Brand uniqueness: When you need a completely distinctive look and feel
Comparison: Bootstrap vs Custom CSS
Factor Bootstrap Custom CSS
Development Speed Fast (pre-built components) Slower (build from scratch)
Learning Curve Moderate (learn framework) Steeper (master CSS)
Design Consistency High (built-in consistency) Variable (depends on skill)
Customization Requires overriding Complete freedom
File Size Larger (unless customized) Smaller (only what's needed)
Browser Compatibility Excellent (thoroughly tested) Variable (depends on testing)
Community Support Large community General CSS community
