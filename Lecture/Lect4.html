HTML Lists

HTML lists help organize information into documents readers can quickly understand, scan, and extract.

Ordered Lists <ol>

Ordered lists are numbered lists where each item is numbered sequentially:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Result:

  1. First item
  2. Second item
  3. Third item

List Style Types

You can customize the numbering style with CSS:

ol {
  list-style-type: upper-roman;
}

Result:

  1. Notebook
  2. Ballpen
  3. Yellow Paper
list-style-type Description
decimal Numbered with numbers (default)
upper-alpha Numbered with uppercase letters
lower-alpha Numbered with lowercase letters
upper-roman Numbered with uppercase roman numbers
lower-roman Numbered with lowercase roman numbers

Unordered Lists <ul>

Unordered lists are bulleted lists:

<ul>
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ul>

Result:

  • Notebook
  • Ballpen
  • Yellow Paper

List Bullet Types

Value Description
disc Bullet (default)
circle Circle
square Square
none No marker

Description Lists <dl>

Description lists are used for terms and their definitions:

<dl>
  <dt>Notebook</dt>
  <dd>- a small book with blank or ruled pages</dd>
  <dt>Ballpen</dt>
  <dd>- a pen that dispenses ink over a metal ball</dd>
</dl>

Result:

Notebook
- a small book with blank or ruled pages
Ballpen
- a pen that dispenses ink over a metal ball

Nested Lists

Lists can be nested inside other lists:

<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

Result:

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Horizontal List with CSS

Lists can be styled horizontally to create navigation menus:

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333333;
}
li {
  float: left;
}
li a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}
li a:hover {
  background-color: #111111;
}

Result:

  • Home
  • News
  • Contact
  • About

HTML Links

Hyperlinks allow users to navigate between web pages:

Basic Link Syntax

<a href="https://www.example.com">Visit Example</a>

Absolute vs Relative Links

Type Description Example
Absolute URL Links to a different website <a href="http://google.com">Google</a>
Relative URL Links within the same website <a href="about.html">About</a>

Link States

Links can be styled differently based on their state:

a:link { color: blue; }       /* Unvisited link */
a:visited { color: purple; }  /* Visited link */
a:hover { color: red; }       /* Mouse over link */
a:active { color: green; }    /* Selected link */

Target Attribute

Controls where the linked document opens:

Value Description
_blank Opens in new window/tab
_self Opens in same window (default)
_parent Opens in parent frame
_top Opens in full body of window

Image as Link

Images can be used as clickable links:

<a href="home.html">
  <img src="logo.png" alt="Home">
</a>

Internal Links

Links to specific sections within the same page:

<a href="#section1">Go to Section 1</a>
...
<h2 id="section1">Section 1</h2>

HTML Images

Images enhance the design and appearance of web pages:

Image Formats

Format Best For Features
WebP All web images Modern format, good compression
GIF Simple graphics, animations Supports transparency, animation
JPEG Photographs Good compression, millions of colors
PNG Graphics with transparency Lossless compression, alpha transparency

Image Syntax

<img src="image.jpg" alt="Description">

The alt attribute provides alternative text for accessibility.

Image Size

Set image dimensions with the style attribute:

<img src="panda.png" alt="Panda" style="width:500px;height:400px;">

Images from Different Locations

Same folder: <img src="logo.png" alt="Logo">

Subfolder: <img src="images/logo.png" alt="Logo">

External server: <img src="https://example.com/logo.png" alt="Logo">

Floating Images

Images can float next to text:

<p>
  <img src="bear.png" alt="Bear" style="float:right;width:100px;height:100px;">
  This text will wrap around the image on the left.
</p>

Image Maps

Create clickable areas within an image:

<img src="worldmap.jpg" alt="World Map" usemap="#worldmap">

<map name="worldmap">
  <area shape="rect" coords="100,50,200,150" href="europe.html" alt="Europe">
  <area shape="circle" coords="300,200,50" href="asia.html" alt="Asia">
</map>

Background Images

Set images as background for elements:

body {
  background-image: url('background.jpg');
}
