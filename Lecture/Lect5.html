Introduction to CSS3

Cascading Style Sheets (CSS) were created to provide a consistent look across many webpages and to separate structure from presentation.

History of CSS

  • 1996: W3C published CSS Level 1
  • 1998: CSS Level 2 published
  • 2000s: Work began on CSS Level 3 (CSS3) as modular specifications
  • 2011: CSS 2.1 recommendation published
Note: CSS3 is backward compatible with CSS 2.1 and includes over 50 modules that can be published independently.

Why CSS Solved a Big Problem

Before CSS, HTML contained tags for formatting (like <font>), which made website development difficult and expensive. CSS separated style formatting from HTML structure.

CSS Syntax and Structure

A CSS rule-set consists of a selector and a declaration block:

selector {
  property1: value;
  property2: value;
}

Explanation:

  • Selector: Points to HTML element(s) to style
  • Declaration block: Contains one or more declarations
  • Declaration: Property:value pair (always ends with semicolon)

CSS Selectors

Element Selector

Selects elements based on element name:

p {
  text-align: center;
  color: red;
}

ID Selector

Selects one unique element based on id attribute:

#para1 {
  text-align: center;
  color: red;
}

Class Selector

Selects elements with a specific class attribute:

.center {
  text-align: center;
  color: red;
}

Grouping Selectors

Apply the same styles to multiple selectors:

h1, h2, p {
  text-align: center;
  color: red;
}

Inserting CSS

There are three ways to insert CSS styles:

1. External Style Sheet

Link to an external .css file (best for site-wide styling):

<head>
  <link rel="stylesheet" href="styles.css">
</head>

2. Internal Style Sheet

Define styles in the <style> element (for single-page styling):

<head>
  <style>
    body {
      background-color: lightblue;
    }
    h1 {
      color: navy;
    }
  </style>
</head>

3. Inline Styles

Apply styles directly to an element (for one-off styling):

<h1 style="color:blue;margin-left:30px;">Heading</h1>

Cascading Order

When multiple styles are defined, they cascade in this order (highest to lowest priority):

  1. Inline styles
  2. Internal style sheets
  3. External style sheets
  4. Browser defaults

CSS Backgrounds

CSS provides several properties for styling element backgrounds:

body {
  background-color: lightblue;
  background-image: url("bg-image.jpg");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: fixed;
}

Shorthand Background Property

body {
  background: lightblue url("bg-image.jpg") no-repeat right top fixed;
}

CSS Borders

CSS border properties allow you to specify style, width, and color of borders.

Border Styles

dotted
dashed
solid
double
groove
ridge

Border Width and Color

p {
  border-width: 5px;
  border-color: red;
  border-style: solid;
}

Individual Side Borders

p {
  border-top: 5px dotted blue;
  border-right: 3px dashed green;
  border-bottom: 4px double red;
  border-left: 2px solid purple;
}

Border Shorthand

p {
  border: 5px solid red;
}

Rounded Borders

p {
  border: 2px solid red;
  border-radius: 10px;
}
